############################################ Disease Prediction Project ################################################################

## Overview

     This project implements a disease prediction application using machine learning. The application takes symptoms as input and predicts a potential disease based on a trained Random Forest Classifier.



## Contents

- `DATASET_.csv`: CSV file containing the dataset used to train the machine learning model.
- `myproject.py`: Python script containing the code for the disease prediction application.
- `requirements.txt`: Text file listing the required Python libraries and their versions.



## Usage

1. Install the required dependencies using.

2. Run the 'myproject.py' script.

3. Enter symptoms when prompted, and the application will predict the potential disease.



## Dataset

The dataset (`dataset.csv`) consists of symptoms and corresponding diseases. Each row represents a set of symptoms and the associated disease.



## Requirements

- Required Python libraries are listed in `requirements.txt`.




## Important Notes
- The predictions are based on machine learning and data collected from the internet.
