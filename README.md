# Disease_prediction_using_ML

This is a simple disease prediction application built using machine learning. The application predicts the disease based on the symptoms provided by the user.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

- Python 3
- pip

### Installing

1. Clone the repository:

```
git clone https://github.com/abhinavpathania/disease-prediction-app.git
```

2. Navigate to the project directory:

```
cd disease-prediction-app
```

3. Install the required Python packages:

```
pip install -r requirements.txt
```

### Usage

1. Run the following command to start the application:

```
python disease_prediction_app.py
```

2. Enter the symptoms separated by commas in the input field.
3. Click on the "Predict" button to see the predicted disease.
